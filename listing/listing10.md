### Вопрос

Что выведет данная программа и почему?




```go
func update(p *int) {
    b := 2
    p = &b
}

func main() {
    var (
        a = 1
        p = &a
    )
    fmt.Println(*p)
    update(p)
    fmt.Println(*p)
}
```

### Ответ

Вывод:
```go
1
1
```

Объяснение:

в Go все передается в функции по значению, даже указатели,
в данном случае внутри функции мы обновили копию указателя, а не исходный,
поэтому значение не изменилось.
